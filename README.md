# Проект: АИ-ассистент для компании ЭПРОТУЛ
вот ссылка на проект t.me/airprotool_bot

## Описание проекта
АИ-ассистент разработан для компании **ЭПРОТУЛ**, специализирующейся на продаже ручного пневматического инструмента. Цель проекта — автоматизировать взаимодействие с клиентами, повысить эффективность обработки запросов и упростить процесс записи на встречи.

Ассистент способен:
1. Вести осмысленный диалог с клиентами с помощью нейросети.
2. Отвечать на вопросы на основе базы знаний компании.
3. Записывать клиентов на встречи и автоматически добавлять события в Google Календарь.

---

## Функционал

### 1. Осмысленное ведение диалога
- Ассистент использует современные технологии обработки естественного языка (NLP) для понимания запросов клиентов.
- Способен поддерживать диалог, уточнять детали и предлагать решения в зависимости от контекста.

### 2. Ответы на вопросы по базе знаний
- База знаний содержит информацию о продукции компании (спецификации, области применения, цены), а также общие рекомендации по использованию пневматического инструмента.
- Ассистент анализирует запрос клиента и предоставляет точные ответы, основанные на данных из базы знаний.

### 3. Запись на встречу и интеграция с Google Календарь
- Клиент может запросить запись на встречу через диалог с ассистентом.
- Ассистент предлагает доступные временные слоты, уточняет детали и автоматически добавляет событие в Google Календарь менеджера.

---

## Используемые сервисы

### 1. Qwen
- **Qwen** был использован для создания системного промта и формирования базы знаний.
- Промт настраивался таким образом, чтобы ассистент мог корректно интерпретировать запросы клиентов и предоставлять релевантные ответы.
- База знаний была структурирована и обучена на основе данных компании ЭПРОТУЛ.

### 2. Suvvy (https://app.suvvy.ai/)
- Платформа **Suvvy** использовалась для создания и настройки ИИ-ассистента.
- Suvvy обеспечивает интеграцию с Google Календарь, что позволяет автоматизировать процесс записи на встречи.
- Интерфейс платформы позволил быстро настроить функционал ассистента и адаптировать его под потребности компании.

---

## Преимущества решения

1. **Автоматизация коммуникаций**: Ассистент работает 24/7, что позволяет обрабатывать запросы клиентов в любое время.
2. **Точность ответов**: База знаний гарантирует, что клиенты получают актуальную и достоверную информацию.
3. **Экономия времени**: Автоматическая запись на встречи и интеграция с Google Календарь освобождают менеджеров от рутинных задач.
4. **Масштабируемость**: Решение легко адаптируется под растущие потребности компании.

---

## Технические требования

1. **Веб-браузер с поддержкой JavaScript**: Для работы с ассистентом необходимо включить JavaScript в браузере.
2. **Доступ к Google Календарю**: Интеграция требует подключения учетной записи Google и предоставления прав доступа.
